# Khởi chạy Dịch vụ Gợi ý (Backend Flask)

Dịch vụ Machine Learning được xây dựng bằng Python (Flask), chịu trách nhiệm huấn luyện mô hình (KNN) và cung cấp API gợi ý phim dựa trên tương tác của người dùng.

### Hướng dẫn cài đặt

1. **Cài đặt Python**

2. **Cài đặt thư viện:**

    ```bash
    pip install pandas flask scipy scikit-learn psycopg2-binary
    ```

3. **Khởi chạy Dịch vụ:**

    ```bash
    python recommender_service.py
    ```

4. **Truy cập:**
    Dịch vụ ML sẽ chạy tại: `http://localhost:5001`
